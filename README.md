### 📝 Project Overview
This repository contains a web blog entry for **"THE DIGITAL VILLAGE: WEB DESIGN COMPETITION"** held at **Bapu Primary-High School**. 

The competition challenged students from Class 7 and 8 to apply their knowledge of **HTML and CSS** to create a digital narrative about rural roots and heritage. Dhanushree's project, "A Glimpse into Paradise: Chikkamagaluru," captures the serene beauty of Karnataka’s coffee capital. It demonstrates how digital storytelling can effectively communicate the tranquil atmosphere of nature and the simplicity of village life.

### 💻 Technical Skills Demonstrated
* **HTML5 Semantic Structure:** Expert use of div-based containers (`blog-box`), hierarchical headers (`h1`, `h2`), and specialized tags like `<b>` and `<i>` for narrative emphasis.
* **CSS3 Styling:** * **Layout Architecture:** Custom design using a centered 60% width container with `margin: auto` and `padding` for a professional blog feel.
    * **Visual UI Design:** Implementation of `border-radius` (10px) on containers and (8px) on images to create a modern, soft-edged aesthetic.
    * **Color Coordination:** Purposeful selection of a Light Pink (`#FFB6C1`) and Sky Blue theme to reflect the "Paradise" concept.
    * **Styling Versatility:** Proficient use of both Internal CSS for global layout and Inline CSS for specific element alignment (e.g., the date and byline).
* **Content Curation:** High-quality descriptive writing focusing on the Western Ghats, coffee estate ecosystems, and the importance of nature-centric retreats.
